### Quick Start Guide - Run Locally

1. **Installation**
```bash
git clone https://github.com/jaektomo/SweepstakeManager.git
cd melbourne-cup-sweepstake
npm install
```

2. **Launch Application**
```bash
npm run dev
```

3. Access your sweepstake manager at `http://localhost:3000`
